Overview
--------

AsyncIO `CyTube <https://github.com/calzoneman/sync>`__ bot

Requirements
------------

-  `Python >=3.4 <https://www.python.org/>`__

Installation
------------

.. code:: bash

    pip install cytube-bot
    pip install cytube-bot[proxy]

.. code:: bash

    git clone https://github.com/dead-beef/cytube-bot
    cd cytube-bot
    pip install -e .[dev]

Building
--------

.. code:: bash

    ./build.sh

Testing
-------

.. code:: bash

    ./test
